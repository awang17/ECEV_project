Instructions for ECEV project
Major directories: 
  /scripts: completed and working scripts to automate process
  /sandbox: files and scripts still in progress
  /CTCF_chipseq: downloaded reference CTCF binding sites from the ENCODE project
  /CTCF_sort_merge: processed CTCF files (sorted by genome position and with overlapping regions merged
  /CTCF_composite: folder with the consensus CTCF binding sequences file (reference for methylome comparison)
